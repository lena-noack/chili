### PACMAN notes for CHILI intercomparison
 18.02.2026

The version of PACMAN used to produce the data 
used for the CHILI protocol paper is stored in 
the /intercomparison/pacman/inputs/model folder 
of this github repository.

These .in files contain the settings used to 
produce the data for each model run, labeled
accordingly. Note the mass of initial free 
oxygen (Init_fluid_O) which has been adjusted 
for each run following the guidelines in the 
protocol paper for models with evolving oxyen 
fugacity.

The README.txt file contains details on how to 
download, install, and run this version of 
PACMAN-P, which slightly differs from PACMAN-P 
Version 1.0 (Krissansen-Totton et al. 2024). 

IMPORTANT: The version of PACMAN used for the
CHILI intercomparison project has been updated
since the version used to produce the data for
the CHILI protocol paper. These updates made 
minor adjustments to the climate model for both
the magma ocean and post magma ocean evolution.

For specific questions regarding any settings 
or values used to produce the output data not 
clarified in README.txt, Krissansen-Totton et 
al. 2024, or the CHILI protocol paper, please 
contact Joshua Krissansen-Totton (joshkt@uw.edu)
