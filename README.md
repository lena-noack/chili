<h1 align="center">CHILI: Coupled atmospHere Interior modeL Intercomparison<br>
(A <a href="https://nexss.info/cuisines/">CUISINES</a> model intercomparison project)
</h1>

> [!IMPORTANT]
> CHILI team email: chili@rockyworlds.org
> 
> CHILI admin email: chili_admin@rockyworlds.org

This is the scripts and data repository for the Coupled atmospHere Interior modeL Intercomparison (<a href="https://nexss.info/cuisines/chili-mip/">CHILI</a>) project. CHILI is an (exo-)planet magma ocean interior-atmosphere model intercomparison project within the <a href="https://nexss.info/cuisines/">CUISINES</a> framework, which aims to provide the exoplanet community with a large and diverse set of comparison and validation of models.

This repository is structured as follows:
- ``protocol`` contains the configs and results of codes from the CHILI protocol paper
- ``intercomparison`` contains the configs and results of codes participating in the main CHILI intercomparison project

Please see the README file inside the ``intercomparison`` folder for instructions on how to submit the output data from your magma ocean code to CHILI. If you want to join CHILI, please first read the [CHILI protocol paper](https://arxiv.org/abs/2511.16142) and then get in touch with chili_admin@rockyworlds.org. We plan and execute CHILI in [this project roadmap](https://github.com/orgs/projectcuisines/projects/1). This is an open community project, so we encourage you to reach out and join the intercomparison!
